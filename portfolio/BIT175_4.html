Rapid Prototype Experimental Build

This is the last rapid build I did at Cascadia. I had three weeks to do it and it was a completely experimental anything goes build. I went with a design for a video game website.

There were a lot of first time efforts putting into a full blown website. This website used Grunt and NPM to compile SCSS as well as the JavaScript put into the page. I also got the chance to do some work with paralax scrolling that not only scrolled the images in the background, but also subtly changed the size of the div tags in each section for the three pages that were built out.

I also added in some quick SCSS animations to the buttons to give them a spring when hovered over. On the villians page there is also a button that gives off a evil laugh when clicked.
