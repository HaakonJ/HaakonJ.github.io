Responsive and non-responsive design assignment description

Client Information

This is a class assignment that I created for BIT 168. The object of the assignment was to make the following webpage into a responisve version of itself.

Project Description

There are several screen shots of the website for you to look at if you want. You can also click the links for a more in-depth look at the sites shown in the images bellow.

As you can see from the screen shots the original website doesn't change to better fit the screen for other screen sizes and the navigation always takes up a certian amount of space on the page. The responsive version of the webpage, however changes the navigation position as well as the content so that it changes layout on smaller screen sizes.
