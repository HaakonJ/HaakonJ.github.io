Sakura-Con Manga Library Database

This was a part of a project to update the excel document that the people in charge of Sakura-Con's Manga Library were using to keep track of what manga was in the library. Sakura-Con is a convention that Seattle has every year in spring designed for anime and manga lovers.

I was on a team of four people in charge of updating the excel spreadsheet over to a MySQL database. I along with several others were responsible for designing all of the prototype models, speaking with the client as well as finally creating the database. I was the main person in charge of creating and updating the prototype database mock-ups in Visio to match the clients needs. I also helped in the final programming of the MySQL database making many of the tables as well as all of the views.

Before this project I had also created an anime database of my own to keep track of the different shows I had seen as well as were planning to watch. So it was a lot of fun to create and it also presented some interesting challenges with planning for the various different relationships required for the library as well as the relationships of the other types of books and magazines that were also in the library.
