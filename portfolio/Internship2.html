City of Lynnwood Web Application

This is some of my most recent work in web design where I was working with a group of two other people to create a treasure hunt application for Lynnwood using geolocation. The initial idea for it was to let you scan for the treasure item within a distance of around 50 feet of you. If you were close enough for it to be scanned it would do so and than apply the change to a users collection of items.

This internship gave me a chance to learn Firebase as well as geolocation and create the JavaScript functions for finding your location and scanning the Firebase database for items near you. I also during the internship got to work adding data to the JSON Tree for firebase both by adding it from the JavaScript and from the Firebase side of things.

This web application also had a collections page, which I provided the functions that would automatically generate the starting user information when signing up that allowed the collection page to generate all the found and unfound items and place the neccesary elements on the page. I also was the one that designed a lot of the assets and prototypes for the web app.

When I wasn'veryt programming and designing for the app I was also communicating back and for between my team and the client. I am thankful to the team I worked with on this as it would have been hard handling all of the authentication, prototyping, programming and designing on my own.
