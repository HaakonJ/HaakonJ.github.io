My History

"I am a 23 year old programmer, tester and web designer."

For the last four years I have in one capacity or another been working web design. Prior to 2014 I have had smaller forays into HTML and Photoshop going back to 2011 with HTML and 2009 with Photoshop, but in a none professional capacity. Beginning in 2014 I started work with Dreamweaver to do actual web design and in 2015 I began using JavaScript and soon after C#. By 2016 I had my first internship remaking part of the Cascadia College website for the Veterans Community.

In 2015, I also began creating small scale databases in Microsoft Access, later in 2016 I would begin using SQL in MySQL and Microsoft SQL Server. In the last year, I have been applying this knowledge in making databases for ASP.NET both with and without MVC. Most Recently I have also made a Firebase Database as part of a second internship with the City of Lynnwood to make a web application that used the Firebase Database as well as Geolocation in conjunction with ArcGIS maps.

Between all of these experiences I have gotten the chance to work on several projects both in a team and solo. Through these experinces I have had the chance to cultivate both team and self managment skills.
